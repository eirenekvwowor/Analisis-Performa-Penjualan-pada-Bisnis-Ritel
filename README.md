# Analisis-Performa-Penjualan-pada-Bisnis-Ritel
Analisis Performa Penjualan pada Bisnis Ritel

Di tengah persaingan bisnis ritel yang semakin ketat, pemahaman terhadap perilaku penjualan dan faktor-faktor yang memengaruhinya menjadi sangat krusial. Perusahaan ritel dituntut untuk mampu mengambil keputusan yang tepat terkait strategi harga, diskon, dan pengelolaan produk agar tetap kompetitif sekaligus menjaga profitabilitas.
Project ini berfokus pada analisis performa penjualan bisnis ritel menggunakan public dataset dari Kaggle sebagai simulasi data perusahaan ritel. Tujuan utama analisis adalah mengidentifikasi pola, hubungan antar variabel, serta insight yang dapat mendukung pengambilan keputusan strategis dan meningkatkan performa bisnis
